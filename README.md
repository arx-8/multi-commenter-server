# multi-commenter-server

## Dev

```sh
npm i
npm start
```

## Usage

### Local execute

```sh
npm start
```

### Call API

```http
POST https://multi-commenter-server.unubo.app/
Content-Type: application/json

{
    "path": "/auth/create",
    "callback_url": "http://localhost"
}
```
